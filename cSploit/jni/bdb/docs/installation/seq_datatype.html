Sequence Datatype Changes

The datatype for several pieces of information associated with sequences has changed from int32_t to u_int32_t:

  • DB_SEQUENCE->get() delta parameter
  • DB_SEQUENCE->set_cachesize() size parameter
  • DB_SEQUENCE->get_cachesize() sizep parameter
  • st_cache_size field of the DB_SEQUENCE_STAT structure
