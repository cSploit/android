Updated SQLite Version

Berkeley DB's SQL interface includes code from SQLite. The version of SQLite used has been upgraded, so that Berkeley DB SQL is now compatible with SQLite version 3.7.16.2. Please see the release notes at http://sqlite.org/changes.html for further information.
