Hot Backup Changes

Because non-UNIX systems do not support atomic file system reads, the db_hotbackup utility has been modified to read data through the environment. If your application is running on a UNIX based system such as Solaris, HPUX, BSD or Mac OS, you can specify the -F flag to read directly from the filesystem. Please refer to Recovery procedures in the Berkeley DB Programmer's Reference Guide for more information on safely backing up your databases.
