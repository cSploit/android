Changes to the build_windows Folder

Visual Studios projects in the build_windows folder have been moved into two new sub folders build_windows/VS8 and build_windows/VS10. Visual Studios 2010 projects have been moved into the folder VS10, and Visual Studios 2008 projects have been moved into the folder VS8.
