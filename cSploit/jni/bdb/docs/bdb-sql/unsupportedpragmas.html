Unsupported PRAGMAs

The following PRAGMAs are not supported by the BDB SQL interface.

PRAGMA journal_mode
PRAGMA legacy_file_format

Also, PRAGMA fullfsync is always on for the BDB SQL interface. (This is an issue only for Mac OS X platforms.)
