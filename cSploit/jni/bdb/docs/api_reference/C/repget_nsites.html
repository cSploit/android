DB_ENV->rep_get_nsites()

#include <db.h>

int
DB_ENV->rep_get_nsites(DB_ENV *env, u_int32_t *nsitesp);

The DB_ENV->rep_get_nsites() method returns the total number of participant sites in the replication group. For Base API applications, his value is configurable using the DB_ENV->rep_set_nsites() method. For Replication Manager applications, this value is determined dynamically.

For Base API applications, this method may be called at any time during the life of the application. For Replication Manager applications, this method may be called only after a successful call to the DB_ENV->repmgr_start() method.

The DB_ENV->rep_get_nsites() method returns a non-zero error value on failure and 0 on success.

Parameters

nsitesp

The DB_ENV->rep_get_nsites() method returns the total number of participant sites in the replication group in nsitesp.

Class

DB_ENV
