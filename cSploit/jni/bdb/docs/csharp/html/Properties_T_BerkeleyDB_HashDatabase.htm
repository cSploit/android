AutoCommit
If true, all database modification operations based on this object will be transactionally protected.

BlobDir
The path of the directory where blobs are stored.
BlobThreshold
The threshold value in bytes beyond which data items are stored as blobs.

Any data item that is equal to or larger in size than the threshold value will automatically be stored as a blob.

A value of 0 indicates that blobs are not used by the database.

NonDurableTxns

NoWaitDbExclusiveLock
If true, configure the database handle to obtain a write lock on the entire database. When the database is opened it will immediately throw LockNotGrantedException if it cannot obtain the exclusive lock immediately. If False, configure the database handle to obtain a write lock on the entire database. When the database is opened, it will block until it can obtain the exclusive lock. If null, do not configure the database handle to obtain a write lock on the entire database.
(Inherited from BaseDatabase.)

Priority

ReadOnly

ReadUncommitted
